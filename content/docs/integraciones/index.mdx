## ¿Cómo funcionan las integraciones?

Cada integración requiere que tengas una cuenta activa en el servicio externo. SkyTienda se conecta mediante **credenciales seguras** (llaves API) que tú proporcionas.

<Callout type="info">
  Las credenciales que ingresas se almacenan de forma encriptada y solo se usan para comunicarse con el servicio correspondiente.
</Callout>

## Acceder a las integraciones

1. Ve a **Configuración → Integraciones**
2. Selecciona el servicio que deseas conectar
3. Sigue las instrucciones específicas de cada integración
