Skytienda usa **WhatsApp Cloud API** de Meta para conectar tu número. Esto permite que Celestino:
- Reciba mensajes de tus clientes
- Responda automáticamente 24/7
- Envíe imágenes de productos
- Procese pedidos y pagos
- Envíe confirmaciones y actualizaciones

## Requisitos previos

Antes de comenzar, asegúrate de tener:

### 1. Cuenta de Facebook

Una cuenta de Facebook activa. Esta cuenta se usará para crear o vincular un Meta Business Manager.

### 2. Número de teléfono

- Debe poder recibir códigos de verificación por **SMS o llamada**
- **No puede estar registrado** actualmente en Whatsapp o Whatsapp Business
- Si ya está en uso, deberás eliminarlo primero (ver sección de solución de problemas)
- Números cortos (short codes) no son permitidos
- Una vez conectado a Cloud API, **no podrás usar ese número** en la app de Whatsapp personal o Business

> [Documentación oficial de números de teléfono - Meta Developers](https://developers.facebook.com/docs/whatsapp/cloud-api/phone-numbers)

### 3. Tu catálogo de Skytienda configurado

Tu [catálogo web de Skytienda](/empezar/catalogo-web) funciona como sitio web de tu negocio durante el registro con Meta. Antes de conectar Whatsapp, asegúrate de tener:

- **Nombre de la tienda** configurado
- **Productos subidos** al catálogo
- **Tienda personalizada** (logo, colores, descripción)

## Proceso de conexión

### 1. Iniciar el proceso

1. Ve a **Configuración → Whatsapp** en tu panel de Skytienda
2. Haz clic en **Conectar Whatsapp**
3. Se abrirá una ventana de Meta para iniciar sesión con Facebook

### 2. Aceptar términos

Revisa y acepta los términos de servicio de Meta para WhatsApp Cloud API.

### 3. Crear o seleccionar Business Manager

Meta requiere un **Business Manager** (también llamado portafolio comercial) para gestionar tu cuenta de Whatsapp Business:

- Si ya tienes uno, selecciónalo de la lista
- Si no tienes, créalo durante el proceso ingresando:
  - Nombre del negocio
  - Dirección
  - URL de tu catálogo de Skytienda o sitio web

### 4. Configurar cuenta de Whatsapp Business (WABA)

1. Selecciona **"Crear nueva cuenta"** o usa una existente
2. Asigna un nombre a tu cuenta WABA
3. Configura la información que verán tus clientes:
   - **Nombre mostrado** (ver reglas abajo)
   - **Categoría** de negocio

### 5. Registrar tu número

1. Ingresa el número de teléfono que usarás
2. Elige cómo recibir el código de verificación:
   - **SMS** (opción predeterminada)
   - **Llamada de voz** (recomendado si el SMS no llega)
3. Ingresa el código de 6 dígitos

<Callout type="warning">
  **Importante:** Si el número ya está en Whatsapp personal o Business, deberás eliminarlo primero. Esto desvinculará el número de cualquier cuenta anterior.
</Callout>

### 6. Confirmación

Una vez verificado el número:
1. Meta procesará la conexión (puede tomar unos segundos)
2. Verás un mensaje de éxito
3. El estado en Skytienda cambiará a **"Conectado"**

## Reglas para el nombre mostrado

El **nombre mostrado** (display name) es lo que verán tus clientes cuando les escribas. Meta tiene reglas estrictas que debes cumplir:

### El nombre DEBE:

- Representar claramente tu negocio, producto, servicio o departamento
- Tener **mínimo 3 caracteres**
- Coincidir **exactamente** con cómo aparece tu marca en tu catálogo de Skytienda o sitio web (incluyendo mayúsculas y espacios)
- Tener relación clara con tu negocio (verificable en tu catálogo de Skytienda o sitio web)

### El nombre NO puede:

- Ser un término genérico (ej. "Moda", "Tienda", "Ropa")
- Ser solo una ubicación geográfica (ej. "Bogotá", "Colombia")
- Incluir palabras como "Oficial" o "Verificado" (implican verificación de Meta)
- Contener nombres de productos de Meta (Whatsapp, Facebook, Instagram, Messenger)
- Usar **solo mayúsculas** (a menos que sea un acrónimo conocido como IBM o NASA)
- Incluir caracteres especiales, emojis o símbolos de marca registrada
- Ser el nombre completo de una persona
- Ser un eslogan o descripción larga
- Tener formato de URL (ej. "mitienda.com")

### Ejemplos

| Incorrecto | Correcto |
|--------------|-------------|
| SUNNY'S CLOTHING | Sunny's Clothing |
| Tienda Oficial | Mi Tienda ABC |
| Moda | Boutique María |
| Fashion NYC | NYC Fashion House |
| Juan Pérez | Ferretería Pérez |

### Requisito de presencia web

El nombre **no puede ser aprobado sin una presencia web verificable**. Meta revisará que el nombre coincida con:
- Tu catálogo de Skytienda, o
- Tu sitio web (con HTTPS)

## Cuándo se muestra el nombre en lugar del número

Para que tus clientes vean tu **nombre** en lugar del número de teléfono, necesitas:

1. Nombre mostrado en estado **"Aprobado"**
2. Negocio [**verificado** en Meta Business Manager](https://es-la.facebook.com/business/help/2058515294227817)
3. Historial activo de mensajes enviados
4. Mantener **alta calidad** en tu cuenta (sin reportes de spam)

## Estados de conexión

| Estado | Significado |
|--------|-------------|
| **No conectado** | No has iniciado el proceso |
| **Pendiente** | La conexión está en proceso |
| **Conectado** | Whatsapp funciona correctamente |
| **Error** | Hubo un problema, revisa los detalles |

## Información del número conectado

Una vez conectado, en tu panel verás:

- **Número** - El número de Whatsapp conectado
- **Nombre mostrado** - El nombre que ven tus clientes
- **Estado de verificación** - Si tu negocio está verificado por Meta
- **Calificación de calidad** - La calidad de tus mensajes según Meta

## Solución de problemas

### "El número ya está en uso"

Si el número está registrado en Whatsapp personal o Business:

1. Abre Whatsapp en el teléfono donde está registrado
2. Ve a **Configuración → Cuenta → Eliminar cuenta**
3. Confirma la eliminación
4. Espera al menos **5 minutos**
5. Intenta de nuevo en Skytienda

### "No recibo el código de verificación"

- Verifica que el número puede recibir SMS internacionales
- Intenta con **verificación por llamada** en lugar de SMS
- Asegúrate de que el número no esté bloqueando mensajes de números desconocidos

### "Nombre mostrado rechazado"

Meta puede rechazar el nombre si:

- No coincide con tu presencia en línea (catálogo de Skytienda, sitio web)
- Parece genérico o spam
- Usa solo mayúsculas
- Contiene caracteres especiales o emojis
- Viola las políticas de nombres

**Solución:** Usa exactamente el mismo nombre que configuraste en tu tienda de Skytienda, respetando mayúsculas y espacios.

### "Business Manager rechazado"

Meta puede rechazar la creación del Business Manager si:

- El nombre del negocio no es claro o parece spam
- La URL del sitio web o catálogo no funciona
- La información proporcionada no coincide entre fuentes

**Solución:** Verifica que toda la información sea coherente entre tu catálogo de Skytienda, sitio web y registro.

### "Error de conexión"

Si el proceso falla sin mensaje específico:

1. Cierra la ventana y vuelve a intentar
2. Asegúrate de usar un navegador actualizado
3. Desactiva extensiones que puedan bloquear popups o bloqueadores de anuncios
4. Contacta a soporte si el problema persiste

## Desconectar tu número

Si necesitas desvincular tu número de Skytienda, consulta la guía de [desconexión de Whatsapp](/empezar/conectar-whatsapp/desconectar-whatsapp). Ten en cuenta que al desconectar un número de la Cloud API se pierde el nombre mostrado, historial y plantillas.
